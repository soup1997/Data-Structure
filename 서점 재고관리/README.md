# 기능
1. 재고의 데이터를 저자의 이름 순으로 정렬하여 출력
2. 특정 작가의 작품을 제목 순으로 출력
3. 새로운 서적 정보 입력
4. 특정 서적의 가격 정보 변경
5. 특정 서적의 재고 숫자 변경
6. 재고에 있는 전체 서적의 재고 숫자 출력
7. 재고에 있는 서적들에 대한 전체 가격 출력
